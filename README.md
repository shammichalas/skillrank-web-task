## ✨ Features

- 🔐 Firebase Authentication
- 🍲 Recipe browsing with categories
- 💾 Save favorite dishes locally
- 💬 Elegant contact form with Email.js
- 🎨 Responsive UI with shadcn-ui & Tailwind
- 🔍 Fast filtering and search
- 📱 Mobile-first design
